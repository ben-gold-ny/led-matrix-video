# led-matrix-video
This repository is designed to let a user input any video, alter its resolution to 8x8 pixels, then allow it to be played on an 8x8 LED matrix using aBasys 3 microcontroller.

Due to the binary nature of the LED matrix we used, this repository is meant for videos with only two colors that can be easily converted into binary logic. This iteration of the project is demonstrated with the music video for "Bad Apple", as it is entirely black and white.
